# TextView

Просмотр многострочного текста на базе `form` (ncurses forms).

## Назначение

* Прокрутка текста внутри области фиксированного размера.
* Не предназначен для редактирования пользователем (в текущей реализации — только скролл).

## Управление

* `Up/Down` — построчно
* `PgUp/PgDn` — постранично
* `Home` — в начало
* `End` — в конец

## Отрисовка

* В зависимости от состояния выбирает:

  * `TextViewInactive` если `!enabled`
  * `TextViewActive` если `focused`
  * `TextView` иначе
* Если включена рамка — рисуется box с активной/неактивной границей.

## Пример

```d
auto tv = new TextView(1, 1, 40, 10, "Log output...");
tv.append("next line");
tv.set("replace all");
```
